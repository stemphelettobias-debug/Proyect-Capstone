Proyecto Capstone
Sistema que consume datos de un Dispositivo IoT mediante una API REST, los procesa con Node - RED y los muestra en tiempo real contenerizado en Docker.

¿Cómo levantar el contenedor con: docker-compose?

Abrir en la terminal:
```bash
   docker compose up -d
```
Accedé al editor de Node-RED en:
http://localhost:1880

Accedé al dashboard con la visualización en:
http://localhost:1880/ui
